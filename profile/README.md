## Activity Rules
1. 아침(오전)에 풀고 싶은 문제 1개씩 선정(백준 실버3~골드4, 프로그래머스 lv.1 부터)
2. 날짜로 별로 브랜치 생성
3. 각자 clone해서 문제 풀이 진행해서 commit
4. 디렉토리 구조 사용알고리즘/문제번호(백준은 B 붙여서 명시, 프로그래머스는 P 붙여서 명시)/문제번호_이름.py
5. 참여가 불가능할 경우 전날 연락
